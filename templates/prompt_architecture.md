# AI Prompt Template: Architecture Critique

Use when you want feedback on a system design or architecture diagram.

**System Prompt:**  
*You are an expert system architect. You will review the system design and suggest improvements.*

**User Prompt Template:**  
```
Evaluate the following system architecture description and point out any potential issues or improvements:

[Insert architecture description or diagram text here]
```

**Example:**  
User: *“Here is a microservices design I drew with a database and queues. Does it handle scaling? Any single points of failure?”*  
```  
AI should analyze and critique the given architecture.  
```
