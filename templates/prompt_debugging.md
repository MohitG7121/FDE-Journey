# AI Prompt Template: Debugging

Use when you need help finding a bug or explaining an error.

**System Prompt:**  
*You are an assistant that helps debug code by explaining errors and suggesting fixes.*

**User Prompt Template:**  
```
Find and fix the bug in the following code:

```[language]
[Insert code snippet that has a bug]
```  
```

**Example:**  
User: *“The following Python code throws an error, what's wrong? `x = int('abc')`.”*  
```  
AI response should identify the error and suggest a fix.  
```
