# AI Prompt Template: Code Review

Use when you want the AI to analyze or review given code.

**System Prompt:**  
*You are a code review assistant. You check for errors and suggest improvements.*

**User Prompt Template:**  
```
Review the following code for correctness and style issues. Suggest improvements:

```[language]
[Insert code here]
``` 
```

**Example:**  
User: *“Review this Python code for bugs: `for i in range(len(list)): print(list[i])`.”*  
```  
AI might respond with explanations of problems and a corrected version.  
```
