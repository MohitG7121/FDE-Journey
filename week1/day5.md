# Day 5: Prompt Engineering (1h)

- **Topics:** Fundamentals of crafting prompts for LLMs.  
- **Resources:** OpenAI Prompt Engineering Guide. Article: *“The Art and Science of Talking to LLMs”*. YouTube: *Prompt Engineering Tutorial* (e.g. Andrew Ng’s intro, or Microsoft).  
- **Exercise:** Experiment with a chat model. Formulate a few prompts (zero-shot, one-shot, chain-of-thought) and observe differences. For example, try a simple math question or JSON output prompt using [System] and [User] roles.  
- **Why:** Effective prompts guide LLMs to give the output you need. OpenAI defines prompt engineering as “writing effective instructions for a model, so it consistently generates content that meets your requirements”. Good prompts include context, instructions, and examples to reduce errors.
