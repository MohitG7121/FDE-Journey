# Day 1: Setup Development Environment (1h)

- **Install Tools:** Python (3.11+), Git, Docker, and an IDE (VS Code or PyCharm).  
- **Verify:** Run `python3 --version`, `git --version`, `docker --version` to confirm installations.  
- **Configure Git:** Set your Git username/email (`git config --global user.name "Your Name"`).  
- **Exercise:** Create a new GitHub repo (e.g. `FDE-Journey`), clone it, add a file `hello_world.py` with `print("Hello, FDE!")`, and commit (`git add`, `git commit`).  
- **Why:** These tools enable scripting, version control, and containerization. Python’s syntax and libraries make it ideal for rapid AI development. Git is a distributed version control system (every developer has a full local history). Docker ensures consistent deployment environments.  
- **Resources:** Official docs: [Git (Atlassian guide)](https://www.atlassian.com/git), [Python Tutorial](https://docs.python.org/3/tutorial/), [Docker Install Guide](https://docs.docker.com/engine/install/). YouTube: e.g. Corey Schafer’s Python/Git setup tutorial.
