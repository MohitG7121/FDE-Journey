# Day 4: FastAPI Basics (1h)

- **Topics:** Getting started with FastAPI, a modern Python web framework.  
- **Resources:** [FastAPI Official Tutorial](https://fastapi.tiangolo.com/tutorial/) (First Steps). Article: *“FastAPI: The Modern Python Web Framework…”*. YouTube: FastAPI crash course (e.g. *Corey Schafer or Tech With Tim*).  
- **Exercise:** Create a simple FastAPI app with one endpoint: `GET /ping` that returns `{"ping": "pong"}`. Run with Uvicorn and open the interactive docs at `/docs`.  
- **Why:** FastAPI is a modern, high-performance API framework for Python. It uses async support and type hints, automatically generates Swagger docs, and lets you build APIs quickly and safely.
