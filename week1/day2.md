# Day 2: Python – Functions and Classes (1h)

- **Topics:** Python functions (`def`, parameters, return) and basic classes (OOP).  
- **Resources:** Python Official Tutorial – [Defining Functions](https://docs.python.org/3/tutorial/controlflow.html#defining-functions) and [Classes](https://docs.python.org/3/tutorial/classes.html) sections. Article: *“Writing Functions in Python—A Beginner’s Guide”*. YouTube: Python OOP tutorial (e.g. *freeCodeCamp*).  
- **Exercise:** Write a function `fibonacci(n)` to return the n-th Fibonacci number. Then create a class `Counter` with methods `increment()` (adds 1) and `value()` (returns current count).  
- **Why:** Functions encapsulate reusable code blocks and improve modularity. Classes bundle data and behavior into objects; e.g. a class instance holds state and methods operate on it.
